# IoT Health Frontend

This is the frontend application for the IoT Health project, built with [React](https://react.dev/) and [Vite](https://vitejs.dev/). It provides a user interface to display health and sensor data, utilizing libraries like Recharts for data visualization.

## Prerequisites
- [Node.js](https://nodejs.org/) (v16 or higher recommended)
- npm (comes with Node.js)

## Setup & Installation

1. Navigate to the frontend directory:
   ```bash
   cd iot-health-frontend
   ```

2. Install the necessary dependencies:
   ```bash
   npm install
   ```

## Running the Application

To start the development server:
```bash
npm run dev
```
By default, the Vite development server will be available at `http://localhost:5173`.

## Building for Production

To build the project for production:
```bash
npm run build
```
This will generate production-ready static files in the `dist` directory. You can preview the production build using:
```bash
npm run preview
```
